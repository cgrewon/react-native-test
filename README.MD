# React Native app Test

This app is for test purpose.


## Tech stacks:

- Javascript
- React Native (CRA)
- redux
- redux-persist
- redux-thunk
- react-native-table-component
- react-native-async-storage/async-storage



## Install Packages

```script
$ yarn install
```

## Run android

```shell
$ yarn android
```

## Run iOS

```shell
$ yarn ios
```
